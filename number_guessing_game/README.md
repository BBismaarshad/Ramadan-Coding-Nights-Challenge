# Number Guessing Game in Python
Welcome to the **Number Guessing Game**! This is a simple Python game where you have 5 chances to guess a randomly generated number between 50 and 100.

## How the Game Works
1. The game will generate a random number between 50 and 100 (inclusive of 100).
2. You will be prompted to guess the number, and you will receive feedback on whether your guess is too low, too high, or correct.
3. You have **5 attempts** to guess the correct number. If you guess it correctly, you win the game!
4. If you run out of attempts without guessing the correct number, the game will reveal the number and end.

## Prerequisites

- Python 3.x (No additional libraries are needed)

## How to Run

1. Clone the repository to your local machine:

 ```cmd
 git clone https://github.com/BBismaarshad/Ramadan-Coding-Nights-Challenge/number_guessing_game.git
 ```
   
2.Navigate to the folder where the game is saved:

```cmd
cd app
```
3.Run the Python script:

```cmd
python app.py
```
4.Enjoy playing the game!
